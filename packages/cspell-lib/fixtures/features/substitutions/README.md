# Substitution Feature

Substitutions are used to transform text before it is spell checked.

Examples:

| before         | after     |
| -------------- | --------- |
| doesn&apos;t   | doesn't   |
| wouldn&apos;t  | wouldn't  |
| unknown&apos;t | unknown't |

See Issue: [8636](https://github.com/streetsidesoftware/cspell/issues/8636)

```tex
% [Issue #5291](https://github.com/streetsidesoftware/cspell-dicts/issues/5291)

The following should not be an issue: "Fr\'echet"
```
