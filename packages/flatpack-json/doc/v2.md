# Ideas on improving the file format

In its current format, small changes can cause large changes in the text file when it comes to GitDiff.

This is an exploration of possible improvements.

- Change the core format to separate our the strings and keys.

## Format

```yaml
header: Meta data related to the file
data: The formatted data.
keys: Array of arrays that represent the Object keys.
strings: An array of strings or string segments.
```
