# Programming language dictionaries

This page lists all supported programming languages and file types with their corresponding file extensions
that triggers a language check in CSpell.

## Programming Languages

{programmingLangDefinitions.map((lang) => (
  <div key={lang.id} style={{ marginBottom: '2rem' }}>
    <h2>{lang.id}</h2>
    {lang.description && <p>{lang.description}</p>}
    {(lang.filenames || []).length > 0 && (
      <>
        <h6>Filenames:</h6>
        <pre><code>{(lang.filenames || []).join('\n') || 'No filename specified'}</code></pre>
      </>
    )}
    {(lang.extensions || []).length > 0 && (
        <>
            <h6>File Extensions:</h6>
            <pre><code>{(lang.extensions || []).join('\n') || 'No file extension specified'}</code></pre>
        </>
    )}
  </div>
))}
